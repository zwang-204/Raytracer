# TWRay

TWRay is a PBRT-styled raytracer created by Tony (Zitong) Wang. It is built with modern C++.

## Features

### Integrators
- Path Tracing
- Bidirectional Path Tracing
- Volumetric Path Tracing
- Stochastic Progressive Photon Mapping
- Direct Lighting

### Light Source
- Area light
- Directional light
- Environment light
- Point light
- Spot light

### Materials
- Disney material
- Glass material
- Matte material
- Subsurface material
- Uber material
